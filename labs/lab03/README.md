### Развернутое решение:
Переводим дано в двоичн. формат

IP-адрес узла: **11000000.10101000.11001000.10001011**

Исходная маска подсети: **11111111.11111111.11111111.00000000** 

Новая маска подсети: **11111111.11111111.11111111.11100000**

Накладываем исходную маску подсетки в двоичн. формате на ip-адрес узла и получаем mother_net

**1)Количество бит подсети** 

Накладываем на ip-адрес узла в двоичн. формате новую маску подсети в двоичн. формате:

11000000.10101000.11001000.10001011

11111111.11111111.11111111.**111**00000 -> 3 бит из октета. которые взаимствуются у узловой части


**2)Количество созданных подсетей**

Формула для нахождения кол-во подсетей 2^n, где n-это кол-во бит, взаимствующиеся у узловой части. В данном случае, это 3, следовательно 2^3=**8** - кол-во созданных подсетей

**3)Количество бит узлов в подсети** 

Накладываем на ip-адрес узла в двоичн. формате новую маску подсети в двоичн. формате:

11000000.10101000.11001000.10001011

11111111.11111111.11111111.111**00000** -> 5 бит из октета. оставшиеся бит узловой части

**4)Количество узлов в подсети**

Формула для нахождения кол-во узлов 2^k-2, где k-это кол-во бит, оставшиеся у узловой части, 2-кол-во служебных адресов. В данном случае, k- это 5, следовательно 2^5=32-2=**30** - кол-во созданных узлов.

**5)Сетевой адрес этой подсети**

Накладываем на ip адрес узла в двоичн. формате, новую маску подсети в двоичн. формате:

11000000.10101000.11001000.10001011 - > 192.168.200.139

**11111111.11111111.11111111.111**00000 -> 11000000.10101000.11001000.10000000 -> **192**.**168**.**200**.128/27

6)IPv4-адрес первого узла в этой подсети

11000000.10101000.11001000.10000000 -> +1 11000000.10101000.11001000.10000001 -> **192.168.200.129**

7)IPv4-адрес последнего узла в этой подсети

Получив адрес шлюза данной подсетки в пункте 8, отнимаем от адреса 1: 192.168.200.159-1=192.168.200.158/27

8)Широковещательный IPv4-адрес в этой подсети

Заполняем единицами все нули в узловой части: 11000000.10101000.11001000.1000000 -> 11000000.10101000.11001000.100**11111** -> 192.168.200.159
